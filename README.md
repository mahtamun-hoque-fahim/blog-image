# Blog Image Store

Cover images for mahtamunhoquefahim.vercel.app/blog

## Convention
- Drop cover image into `/covers/`
- Name it exactly after the blog post slug (any extension)
- Example: `covers/why-i-moved-to-next16.png`

## Usage in site
```
https://raw.githubusercontent.com/mahtamun-hoque-fahim/blog-image/main/covers/{slug}.{ext}
```
